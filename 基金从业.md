现金流量表(收付实现制)的结构：投资活动产生的现金流量 CFI，筹资活动产生的现金流量 CFF，经营活动产生的现金流量 CFO  
利润表：收入、费用、利润  
资产负债率 = 负债 / 总资产  
权益乘数 = 资产 / 所有者权益 = 1/(1-资产负债率)  
负债权益比 = 负债 / 所有者权益 = 资产负债率/(1-资产负债率)  
利息倍数 = 息税前利润(EBIT) / 利息  
流动比率 = 流动资产 / 流动负债 衡量企业是否有足够的流动资产来偿还流动负债，维持在 2 合适  
速动比率 = (流动资产 - 存货) / 流动负债 衡量企业在不依赖存货变现的情况下，是否能偿还流动负债，维持在 1 以上合适  
年均总资产 = (期初总资产+期末总资产)/2  
总资产周转率 = 年销售收入 / 年均总资产  
年均存货 = (期初存货+期末存货)/2  
存货周转率 = 销售成本 / 年均存货  
存货周转天数 = 365/存货周转率  
应收账款流转率 = 销售收入 / 年均应收账款  
应收帐款周转天数 = 365 / 应收帐款周转率  
销售利润率 = 净利润 / 销售收入  
资产收益率 = 净利润总额 / 总资产  
净资产收益率 = 净利润 / 所有者权益 = 资产收益率 \* (1+负债权益比)  
净利润 = 息税前利润 - 利息 - 税  
杜邦恒等式：净资产收益率 = 销售利润率 \* 总资产周转率 \* 权益乘数  
总资产 = 总负债 + 所有者权益， 所有者权益 = 资产(流动+非流动) - 负债 = 净资产  
所有者权益又称股东权益或净资产，主要包括股本、资本公积、盈余公积、未分配利润  
FV(终值) = PV(现值) \* (1+i)\*\*n，现值是将来货币的现在价值  
实际利率 = 名义利率 - 通膨  
2 年后的 1 年远期利率 = (1+三年利率)\*\*3 / (1+两年利率)\*\*2 - 1

### 5. 固定收益投资

浮动利率 = 基准利率 + 利差  
利率升高，债券价格下降；公司信用下降、债券价格下降，到期利率上升  
当前收益率 = 年息票利息 / 债券当前市场价格。例：每年支付利息 6 元，当前市场价格为 95，则当前收益率为 6/95 = 6.32%  
到期收益率怎么算？ 当前价格 = Sigma(1~n)利息/(1+到期收益率)\*\*n + 到期偿还的本金 / (1+到期收益率)\*\*n 例：面值 100，2 年期，每年付 6 元利息，当前价格 95， 95 = 6/(1+y)+(6+100)/(1+y)\*\*2,结果为 8.836%  
当期收益率和到期收益率关系：债券价格越接近面值，期限越长，当期收益率越接近到期收益率  
到期收益率隐含假设：持有到到期、利息在投资利率不变  
固定利率债券内在价值：n 为期限，C 为利息，M 为面值，r 是市场利率，C/(1+r) + ...+ C/(1+r)\*\*n+M/(1+r)\*\*n  
零息债券内在=到期面值的现值 = 面值/(1+利率)\*\*n  
统一公债内在：利息/利率  
内在价值>票面 就是溢价发行，内在价值是未来现金流的现值总和，零息、固定利率债券有不同的计算方式  
久期： [1×C/(1+r)¹ + 2×C/(1+r)² + ... + n×(C+M)/(1+r)ⁿ] ÷ V  
货币市场工具都是大宗交易，机构玩，个人投资者不玩

### 11. 投资风险管理
